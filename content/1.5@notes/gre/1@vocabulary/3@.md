- 向下
  - sub/sup/sur
    - subvert 颠覆

    - suppress
    - supplant 取代
    - surreptitious 偷偷摸摸的
      - rept 爬
      - reptile 爬行动物

- 向前
  - pre (= before)
    - precursor 先驱，前身
      - curs = run
      - excursion 跑到外面 / 旅行
    - prescient 有先见之明的
      - sci = know
      - science 科学
      - conscious 有意识的
    - predilection 偏好
      - lect = choose，选择
      - elect 选举
      - collect 收集
    - preclude 阻止
      - clud = clus = close
      - include / exclude / conclude
      - seclude 隔开
        - separate
    - premeditate 提前计划
      - meditate 冥想
    - presumption 假定，放肆
      - sumpt = sum = 拿
    - precedent 先例
      - ced = 走
    - presage 预示，预兆
    - predominant 突出的，占优势的
      - dom = 家，控制
      - domestic 国内的
      - dominate 支配
      - indomitable 不屈不挠的，勇敢的
  - pro (= more)
    - proliferation 增生
    - provenance 起源
      - ven = come
    - prominence 明显
      - min 突出
      - eminent 出名的
    - prophetic 预见正确的
    - propitious 吉利的
      - pit = pet = to seek，寻求
    - proscribe 正式禁止
      - scribe = 书写
      - describe 描述
      - prescribe 开处方
    - prognosis 预测，展望

    - probabilistic 基于概率的
    - proclivity 嗜好
    - propensity 习性
    - prototypical 典型
    - provincialism 狭隘
    - provoke 激怒，激起
      - voc = vok = voice
      - vocalization / invoke / evoke
    - prodigious 巨大的
    - profligacy 挥霍
      - flig = flict = strike，打
      - conflict 冲突

    - forestall 抢先

## 循环

- re (again/back/against)
  - recapitulate 总结回顾 (recap)
    - cap = head
    - capital 首都
    - capitulate 投降屈服
  - repudiate 拒绝，否认
    - pud = feel shame
    - impudent 无耻的
  - revitalize 使恢复元气
    - vit = viv = 生命
    - vital 重要的
    - vivid 生动的
    - vigor 活力
  - replicate 复制，重做（不完全一致）
    - duplicate 复制（完全一致）
  - regurgitate 反刍，机械重复
    - gurg = serve
    - > fic = fac = fact，做
  - reverberate 回荡，震荡
    - verb = 声音
  - rejuvenate 使年轻
    - juve = 年轻
    - juvenile 青少年的
  - resent 愤恨不满
    - send = sens = to feel
    - consent 同意
    - dissent 异议

## 中间

  - interrelationship 相互关系
  - interaction 互动，相互作用(interplay)
  - interfere 打扰，影响
    - fere = bring，带来
  - intervene 介入争执
    - ven = 来到
- mid / middle / med
  - midday 中午
  - midnight 午夜
  - immediately 没有中间间隔的，立刻
  - Mediterranean 地中海
    - terr，土地，地球
    - territory 领土
    - terrain 地形
