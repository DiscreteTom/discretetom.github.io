# 总结

## 各问题与算法的时间复杂度与空间复杂度

| 算法与问题 | 时间复杂度 | 空间复杂度 |
| --- | --- | --- |
| 递归分治-二分查找 | O(logn) | - |
| 递归分治-大整数乘法 | O(n^1.59) | - |
| 递归分治-合并排序 | O(nlogn) | O(n) |
| 递归分治-快速排序 | 平均O(nlogn) 最坏O(n^2) | O(n)或O(logn) |
| 递归分治-线性时间选择 | O(n) | - |
| 动态规划-矩阵连乘 | O(n^3) | O(n^2) |
| 动态规划-最长公共子序列 | O(m+n) | - |
| 动态规划-最大子段和 | O(n) | O(n) |
| 动态规划-凸多边形三角剖分 | O(n^3) | O(n^2) |
| 动态规划-0-1背包 | O(cn) c:背包容量;n:物品数 | - |
